About the hack

Using the WebAudio API to slice and rearrange SoundCloud (DOWNLOADABLE) sounds.
Access http://audiojedit.nodester.com/:username/:trackpermalink where /:username/:trackpermalink is the same as the SoundCloud permalink.
Example: http://audiojedit.nodester.com/berlinhunt2/s16-french-accent-scmeetup
From: http://soundcloud.com/berlinhunt2/s16-french-accent-scmeetup

How to use

Select the clip from the blue waveform. When you're done doing the selection that clip will be played.
Press enter to add it to the sequencer. Press the space bar to play it all.

Technology used

  • WebAudio API
  • Node.js
  • Socket.io
  • jQuery
